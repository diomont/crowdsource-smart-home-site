Crowdsourcing Smart Home Data

An anonymous, Open Source, data gathering platform for smart home information

The Goal

The main objective of this project is to create a platform that can gather through a crowdsourcing mechanism anonymous information from homes. This project is based on the Open Source Software project Home Assistant and the Portuguese community of users.

Components

Data Lake

An heterogeneous aggregation of the collected data, receiving the crowdfunded data via an Ingest API and making it available for the dashboard, public use and data dumping through distinct APIs.

Dashboard

A web application to visualize our data lake, with statistics and graphics on our collected info.

Home Assistant

To participate, smart home users may install a component on their Home Assistant, which will gather and send data anonymously according to the user's consent configuration.
